Abstract

This work describes a filtering model based on selective dissemination of information and software agent paradigms. It addresses the information retrieval problem in wide area network messaging environments (the Matrix). Filtering stages based on user preference considerations are developed. New term weighting and phrase generation methods within the vector space indexing and retrieval model are presented and their performance is evaluated; a retrieval effectiveness measure developed specifically for such an environment is used in the evaluation. The definition and motivation for separate filtering stages within an information server are presented. The implementation of a retrieval performance test environment called Pasadena is also presented. This system takes the form of a WAN information subscription service whose subscribers formulate multi-stage filtering functions in their profiles.
